# Sorting Algorithms in C++

A beginner-friendly C++ repository for practicing basic sorting algorithms. Currently, this repository includes a simple Bubble Sort implementation.

## Algorithm Included

* Bubble Sort

## Concepts Used

* Array
* Loop
* Comparison
* Swapping
* Basic sorting logic

## Technologies Used

* C++

## How to Run

```bash
g++ bubble_sort.cpp -o bubble_sort
./bubble_sort
```

## Learning Outcome

This practice helped me understand how sorting works by comparing adjacent elements and swapping them when needed.
